Project Name: HerMarket
Description: Women Entrepreneur E-commerce Platform
Tech Stack:
- Frontend: HTML, CSS
- Backend: Python Flask
- Database: SQLite

Features:
- Seller & Customer Login
- Product Listing
- Customer Dashboard
- Order History
- Verified Women-Owned Products

How to Run:
1. Install Python
2. pip install flask flask-cors
3. python app.py
4. Open http://127.0.0.1:5000/
